# InterviewIQ Frontend

This is the frontend client for the InterviewIQ platform. It is a React application built with Vite for fast bundling and development.

## Stack
- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **Icons**: Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. To build for production:
   ```bash
   npm run build
   ```
